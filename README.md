## Dependencies

- Express
- Node 5.10.x or above
- Bcrypt
- Body-parser
- Chalk
- Cookie-session
- Dotenv
- Ejs
- Morgan
- Node-sass-middleware
- Pg
- Ppg-native
- Twilio

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node run local` command.
